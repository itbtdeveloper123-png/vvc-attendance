សេចក្តីផ្តើម

  3. រូបភាពតំណាង
  4. ជំហានបន្ទាប់គ្រាន់តែ រើសប្រភេទបុគ្គលិក (ជំនាញ / កម្មករ) និងបញ្ជូល ID និងចុចប៊ូតុងចូល
